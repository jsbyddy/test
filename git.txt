mkdir 目录名 //创建目录
cd 目录名 //进入目录
pwd //显示当前目录
git init //创建仓库
git add 文件名（*表示所有文件）//把文件添加到仓库
git commit //把添加到仓库中的文件进行提交
git status //查看状态
git diff //查看修改的内容

提交修改和提交新文件是一样的两步：
第一步是git add 
第二步git commit

git reset --hard HEAD^ //退回到上一个版本
git reset --hard 1094a //退回到指定的版本
git reflog //用来记录每一次命令

git 修改
git checkout -- 文件名 //撤销修改，恢复到最近的版本
